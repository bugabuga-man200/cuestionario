¿Qué es el internet? El internet es una comunidad virtual que puede comunicar todo el mundo y uno puede encontrar información que no encontraría físicamente.
La información se transmite por medio de cables, donde la electricidad e  información viaja de un punto a otro.
La dirección IP es el nombre del dominio y los DNS es el encargado de traducir los numeros de la dirección IP.
Una canción de Spotify o cualquier información que se encuentre en el internet viaja por medio de ondas a tu dispositivo, para que así pueda ser reproducido.
HTTP es un protocolo de acceso para las páginas web a través de Internet y HTTP son las siglas para Hypertext Transfer Protocol que se traduce al español como el “protocolo de transferencia de hipertextos”
Para protegerte de los criminales de internet debes de tener una dirección IP segura y 
no meterte o aceptar cosas que puedan ser potenciales virus..
